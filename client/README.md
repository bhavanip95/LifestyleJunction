# Lifestyle Junction

A category-driven lifestyle content hub. The goal is to organize articles into clear, visual categories and subcategories so users can quickly find what they need across everyday topics like health, food, family, and culture.

## Goal

Build a structured lifestyle portal where:

- Categories act as top-level entry points (cards, routes, hero sections).
- Subcategories provide focused content lanes for discovery.
- The experience feels curated and easy to navigate, with consistent imagery per category and subcategory.

## Tech Stack

- React 19
- Vite 6
- React Router DOM 7
- Tailwind CSS 4
- DaisyUI 5 (UI components)
- ESLint 9
